상세내용


  • 찾고자 하는 Acitity에 대한 각각의 키워드와 날짜로 검색 시 네이버API를 활용하여 가까운 거리 순으로 추천
  • Login : 로그인 (DB와 연동 확인, 로그인 기록 저장)
  • Logout: 로그아웃 (main 페이지로 이동)
  • Register : 회원가입 (사용자에 대한 위도 경도 저장
  • Main : 각종 리뷰, 추천 Activity 구현
  • Travel : 여행관련 동행 페이지(여행 장소, 기간, 인원 등)
  • Dining : 식사관련 동행 페이지 (식사 장소, 기간, 인원 등)
  • Culture/Arts : 문화,예술 관련 동행 페이지 (Activity 장소, 기간, 인원 등)
  • Sports : 스포츠 관련 동행 페이지 (Activity 장소, 기간, 인원 등)
  • Wishlist : 즐겨 찾기 기능과 각 유저의 위시리스트 관리
